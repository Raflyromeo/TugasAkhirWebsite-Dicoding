Selamat Datang di Portfolio Saya

Saya adalah mahasiswa Sarjana semester 5 Universitas Gunadarma yang memiliki ketertarikan besar di bidang Web Development dan Desain Antarmuka. Saya terbiasa membangun aplikasi web modern dengan fokus pada tampilan yang rapi, responsif, serta pengalaman pengguna yang baik.

Fokus Pengembangan

Frontend Development

Membangun antarmuka yang menarik dan responsif

Backend Development

Mengembangkan logika dan database yang solid

UI/UX Design

Merancang pengalaman pengguna yang optimal

Dalam proses pengembangan, saya menggunakan HTML, CSS, JavaScript, React JS, TypeScript, dan Next.js, serta Tailwind CSS untuk mempercepat dan menjaga konsistensi desain. Untuk kebutuhan desain dan perencanaan antarmuka, saya terbiasa menggunakan Figma dan Canva.

Di sisi backend dan pengelolaan data, saya memiliki pengalaman menggunakan PostgreSQL, MySQL, Oracle, dan MongoDB, serta terbiasa bekerja dengan Git dan GitHub dalam pengembangan berbasis versi dan kolaborasi tim.
