# Portfolio Web Developer

Website portfolio pribadi yang menampilkan perjalanan belajar dan pengembangan web. Proyek ini dibuat sebagai bagian dari pembelajaran di **Dicoding Indonesia** dengan fokus pada penerapan **HTML semantic** dan **desain responsif**.

---

## Deskripsi

Website portfolio ini dirancang untuk mendokumentasikan proses belajar web development, menampilkan keterampilan teknis, serta melakukan _showcase_ terhadap proyek-proyek yang telah dikerjakan. Pengembangan website difokuskan pada tampilan yang **responsif**, **user-friendly**, dan **mudah dipahami**.

---

## Fitur

- Desain responsif untuk desktop, tablet, dan mobile
- Navigasi _sticky_ agar menu tetap terlihat saat _scrolling_
- _Profile card_ berisi informasi biodata
- Tampilan _tech stack_ yang terstruktur dan jelas
- _Project showcase_ lengkap dengan deskripsi
- Transisi dan animasi halus pada interaksi
- Integrasi ikon menggunakan **Font Awesome**

---

## Teknologi yang Digunakan

- **HTML5** — struktur dan semantic website
- **CSS3** — styling, layout, dan responsive design
- **Font Awesome 6.4.0** — library ikon

## Cara Menjalankan Project

1. **Clone repository**

   ```bash
   git clone https://github.com/Raflyromeo/TugasAkhirWebsite-Dicoding.git
   ```

2. **Masuk ke folder project**

   ```bash
   cd TugasAkhirWebsite-Dicoding
   ```

3. **Jalankan website**
   - Buka file `index.html` melalui browser, atau
   - Gunakan **Live Server** di Visual Studio Code

---

## Responsive Breakpoints

Website ini menggunakan beberapa _breakpoint_ utama:

- **Desktop** : > 992px
- **Tablet** : 768px – 992px
- **Mobile** : < 768px
- **Small Mobile** : < 480px
